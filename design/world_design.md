# 超级大地图城市建造游戏 — 设计书 v2.0

---

## 一、总体架构

### 1.1 游戏世界层次

```
游戏启动
  │
  ├→ 文明选择界面（中国/英国/埃及/罗马...）
  │
  ├→ 超级大地图 (100,000×100,000) ← 区块按需生成
  │   │
  │   ├→ 大陆 (地球比例，约30%陆地)
  │   ├→ 海洋 (约70%)
  │   ├→ 岛屿 (零星分布，各具特产)
  │   │
  │   └→ 玩家主城（点击进入城市视图）
  │         │
  │         └→ 城市微地图 (80×80) — 现有城市建造系统
  │
  └→ 贸易系统（世界级连线 + 运输动画）
```

### 1.2 数据架构：区块系统（Chunk System）

```
世界地图 100,000 × 100,000
    ↓
划分为 64×64 的区块 (Chunk)
    ↓
每个区块 ≈ 4KB（1字节地形 + 1字节资源）
    ↓
按需生成（Perlin噪声种子驱动），只加载视野内区块
    ↓
≈ 1,563 × 1,563 = 2.4M 区块总量，但仅加载 ~20×20 视野区块
```

| 特性 | 值 |
|------|-----|
| 世界地图尺寸 | 100,000 × 100,000 网格 |
| 区块大小 | 64 × 64 网格 |
| 总区块数 | ≈ 2,441,406 |
| 一次性加载 | ~400 区块 (20×20视野) |
| 生成算法 | 多层 Perlin 噪声 + 种子 |
| 持久化 | 只保存玩家探索过的区块 |

## 二、文明系统

### 2.1 文明列表（首期）

| 文明 | 建筑风格 | 特色产品 | 特性加成 |
|------|---------|---------|---------|
| 🏮 **中国** | 中式飞檐/红墙/琉璃瓦 | 丝绸、瓷器 | 人口增长 +15%，文化影响力 |
| 🏛️ **罗马** | 石柱/拱门/大理石 | 大理石建材、葡萄酒 | 建筑速度 +15%，道路效率 |
| 🏰 **英国** | 哥特式/红砖/尖顶 | 纺织品、工业机械 | 工业产出 +20%，贸易收入 |
| 🕌 **埃及** | 方尖碑/纸莎草柱/金色 | 纸莎草、香料 | 农业产出 +25%，山地资源 |
| ⛩️ **日本** | 唐破风/木结构/瓦顶 | 漆器、武士刀 | 渔业 +30%，岛屿加成 |
| 🏴‍☠️ **维京** | 长屋/木雕/兽皮 | 木材、毛皮 | 海运速度 +25%，军事（预留） |

### 2.2 特色产品机制

每个文明有 2 种**特色产品**：
- **专属特产**：仅该文明可生产（如丝绸仅中国、工业机械仅英国）
- **贸易溢价**：特色产品在其他文明城市售价高 50-200%
- **文化传播**：特色产品出口增加文明影响力

## 三、贸易系统（重量级）

### 3.1 贸易资源类型

| 类别 | 资源 | 产地 | 用途 |
|------|------|------|------|
| 🌾 **基础物资** | 粮食、木材、石材 | 普通农田/森林 | 城市发展基础 |
| ⛏️ **矿产** | 铁矿、铜矿、煤矿 | 山地/矿区 | 工业生产 |
| 🏭 **加工品** | 工具、机械、建材 | 工业区特有建筑 | 城市升级需求 |
| 🎁 **奢侈品** | 丝绸、瓷器、香料、宝石 | 文明特产/岛屿 | 高价值贸易 |
| 💰 **通货** | 金币 | 税收/贸易 | 通用货币 |

### 3.2 贸易路线

```
贸易路线 = 源城市 → 航线/陆路 → 目标城市
            ↓                    ↓
       出口货物             消耗货物
       赚取金币             支付金币 + 获得资源
```

**路线类型：**
- **陆路**：两城间有连续陆地路径，安全但慢
- **海运**：需要港口，快但概率遇海盗
- **混合**：陆路+海运组合

### 3.3 市场价格系统

```
价格 = 基础价 × (1 + 需求系数) × (1 - 供应系数 × 0.3)

- 需求系数：基于目标城市对该资源的需求
- 供应系数：基于源城市该资源的仓库存量
- 全球价格波动：每 tick 随机 ±3%
- 事件影响：海盗、丰收、灾害等事件引起价格剧烈波动
```

### 3.4 贸易运输

- 商队/商船在世界上可见（类似《文明》的商队单位）
- 运输有时间延迟（距离 ÷ 速度 tick）
- 商队可被拦截（海盗/土匪，预留战斗系统）
- 玩家可手动派出/撤回商队

## 四、视图切换机制

### 4.1 默认主城

- 玩家开局时，在出生地自动建立 **主城**
- 主城在世界地图上显示为**文明风格的标志性建筑**（3D 视角下可见城池轮廓）
- **点击主城** → 镜头缩放过渡 → 进入城市微地图
- **城外点击** → 保持在超级大地图

### 4.2 世界地图比例

```
世界地图一个格 = 城市微地图 64×64 像素
主城占据世界地图 3×3 格范围
世界地图渲染：低细节（颜色块标识地形）
```

### 4.3 缩放与过渡

```
世界地图 (zoom=0.001~0.01)
      │ 双击/点击主城
      ↓
过渡动画 ≈ 0.5秒
      │
      ↓
城市微地图 (zoom=0.18~3.0) — 现有摄像机控制
      │ 缩放到最小 + 点击世界空白处
      ↓
返回世界地图
```
